Spectral Bias of (ReLU) MLPs
• Deep Networks prioritize learning simple patterns that usually 
generalize well across data samples
• Prioritizes learning low frequency information
• Learning of high frequency information takes longer time and often deeper 
network architecture
• Solutions
•  Use Fourier feature mapping, i.e., positional encoding
• Paper: Fourier Features Let Networks Learn High Frequency Functions in Low 
Dimensional Domains, Tancik et al.
• Use sinusoidal activation function (SIREN)
On the Spectral Bias of Neural Networks, Rahaman et al.

<!-- Page 34 -->
34
IITK CS661: Big Data Visual Analytics: Soumya Dutta
MLP with Sinusoidal Activation Function
• A good fit for images, 3D field type data that can be periodic and 
contain information of various types of frequencies
• Textures, edges, fine details
• Learning in a rich Fourier spectrum space
• Model can approximate functions with high-frequency components without 
needing deep networks
• Stable Gradients 
• sine function provides stable and informative gradients
• If initialized carefully, gradients do not vanish or explode
